#Преобразовать один массив в другой
1. Принять на вход исходный массив
2. Пройтись по массиву, проверяя количество символов в каждом элементе массива. Отсечь все элементы с 4 и более символами
3. Создать новый массив, включив в него только те элементы, которые содержат 3 либо менее символов
4. Вывести на консоль новым массив 
5. Сохранить все изменения, отправить данные на удаленный репозиторий
